* MS Visual Studio 2012 or higher
* .Net Framework 4.5.1
* MySql
* Git

## Setting up the code ##

+Settings:
* Username = `root`
* Password = `root`

*Run sql script: `TradeHubDBScript.sql` located in `..\tradehub-ui\database\`
